# Emoji Auto-Completion for Markdown
A plugin for Joplin that provides auto-completion for emojis in Markdown files.

## Features
It provides a popup menu with emoji suggestions when typing `:` in a Markdown file. The suggestions are based on the emoji names and can be selected using the arrow keys or mouse.

## Installation
The typical way to install plugins is through the built-in Joplin plugin manager.

* Go to Tools → Options → Plugins
* Search for "Markdown Emoji Autocomplete" in the search box
* Click Install and restart Joplin
